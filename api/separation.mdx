## Overview

Source Separation will enable automatic separation of different audio sources from mixed recordings. This feature can isolate vocals, instruments, or other sound sources, useful for:

- Extracting vocals from music
- Isolating speech from background music
- Karaoke applications
- Audio remixing and production
- Removing background music from voice recordings

## Planned Usage Flow

Once implemented, the typical workflow will be:

```typescript
import {
  initializeSeparation,
  separateSources,
  unloadSeparation
} from 'react-native-sherpa-onnx/separation';

// 1. Initialize separation
await initializeSeparation({
  modelPath: { type: 'auto', path: 'models/separation-model' }
});

// 2. Separate audio sources
const sources = await separateSources('mixed-audio.wav');

console.log(`Separated ${sources.length} sources:`);
for (const source of sources) {
  console.log(`  - ${source.sourceId}: ${source.outputPath}`);
}

// 3. Clean up
await unloadSeparation();
```

## Use Cases

### Extract Vocals for Transcription

Separate vocals from background music before speech recognition:

```typescript
// Planned integration
const sources = await separateSources('song-with-speech.wav');
const vocals = sources.find(s => s.sourceId === 'vocals');

if (vocals) {
  const transcript = await transcribeFile(vocals.outputPath);
  console.log('Lyrics:', transcript.text);
}
```

### Karaoke Application

Remove vocals to create instrumental tracks:

```typescript
const sources = await separateSources('original-song.wav');
const instrumental = sources.find(
  s => s.sourceId === 'instruments' || s.sourceId === 'accompaniment'
);

if (instrumental) {
  console.log('Instrumental track:', instrumental.outputPath);
}
```

### Audio Cleanup

Isolate speech from background noise:

```typescript
await initializeSeparation({
  modelPath: { type: 'auto', path: 'models/speech-separation' }
});

const sources = await separateSources('noisy-recording.wav');
const speech = sources.find(s => s.sourceId === 'speech');

if (speech) {
  // Use clean speech for transcription
  const result = await transcribeFile(speech.outputPath);
}
```

### Batch Processing

Process multiple files:

```typescript
const files = ['song1.wav', 'song2.wav', 'song3.wav'];

await initializeSeparation({
  modelPath: { type: 'auto', path: 'models/separation' }
});

for (const file of files) {
  const sources = await separateSources(file);
  console.log(`${file}: separated into ${sources.length} sources`);
}

await unloadSeparation();
```

## Model Support

The separation feature will support both `.tar.bz2` archive models and standalone `.onnx` models, as indicated by the model download configuration:

```typescript
// From ModelDownloadManager.ts:492

  return ext === 'tar.bz2' || ext === 'onnx';
```
