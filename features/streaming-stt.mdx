## Overview

The **streaming STT** API provides real-time recognition with partial results and endpoint detection. Use it when you feed audio incrementally (e.g. from a microphone) and want low-latency, incremental transcripts.

**Key Features:**
- Real-time recognition with partial results
- Automatic endpoint (end-of-utterance) detection
- Low-latency microphone input support
- Stream-based architecture (one stream per utterance)
- Configurable endpoint detection rules
- Hotwords support (transducer models)

<Note>
  **Import path:** `react-native-sherpa-onnx/stt` (streaming is not exported from the main package)
</Note>

## Quick Start

```typescript
import { createStreamingSTT } from 'react-native-sherpa-onnx/stt';

// 1) Create streaming engine
const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/sherpa-onnx-streaming-zipformer-en-2023-06-26' },
  modelType: 'transducer', // or 'auto' to detect
});

// 2) Create a stream (one per recognition session)
const stream = await engine.createStream();

// 3) Feed audio chunks and get results
const samples = getPcmSamplesFromMic(); // float[] in [-1, 1]
await stream.acceptWaveform(samples, 16000);

if (await stream.isReady()) {
  await stream.decode();
  const result = await stream.getResult();
  console.log('Partial:', result.text);
  
  if (await stream.isEndpoint()) {
    console.log('Utterance ended');
  }
}

// 4) Cleanup
await stream.release();
await engine.destroy();
```

## Convenience API

**One call per chunk** (fewer bridge round-trips):

```typescript
const { result, isEndpoint } = await stream.processAudioChunk(samples, 16000);
console.log(result.text);
if (isEndpoint) console.log('Utterance ended');
```

## Endpoint Detection

Configure when the engine considers an utterance finished:

```typescript
const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/streaming-zipformer' },
  modelType: 'transducer',
  enableEndpoint: true,
  endpointConfig: {
    rule1: {
      mustContainNonSilence: false,
      minTrailingSilence: 1.0,  // 1s silence = end
      minUtteranceLength: 0,
    },
    rule2: {
      mustContainNonSilence: true,
      minTrailingSilence: 0.8,
      minUtteranceLength: 0,
    },
    rule3: {
      mustContainNonSilence: false,
      minTrailingSilence: 0,
      minUtteranceLength: 30,  // max 30s per utterance
    },
  },
});
```

**EndpointRule fields:**
- `mustContainNonSilence` (boolean): If true, rule only matches when segment contains speech.
- `minTrailingSilence` (number): Min trailing silence in seconds.
- `minUtteranceLength` (number): Min (or max, depending on rule) utterance length in seconds.

## Checking Streaming Support

Use **`getOnlineTypeOrNull(detectedType)`** to check if a detected STT model type supports streaming:

```typescript
import { getOnlineTypeOrNull, createStreamingSTT, detectSttModel } from 'react-native-sherpa-onnx/stt';

const detected = await detectSttModel({ type: 'asset', path: 'models/zipformer' });
const onlineType = getOnlineTypeOrNull(detected.modelType);

if (onlineType !== null) {
  // Model supports streaming
  const engine = await createStreamingSTT({
    modelPath: { type: 'asset', path: 'models/zipformer' },
    modelType: onlineType,
  });
} else {
  // Model is offline-only (e.g. Whisper)
  console.log('This model does not support streaming');
}
```

## Microphone Integration

For **live microphone capture** with automatic resampling, use the PCM Live Stream API:

```typescript
import { createStreamingSTT } from 'react-native-sherpa-onnx/stt';
import { createPcmLiveStream } from 'react-native-sherpa-onnx/audio';

const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/streaming-zipformer' },
  modelType: 'transducer',
});
const stream = await engine.createStream();

// Start microphone capture (auto-resamples to 16kHz)
const mic = await createPcmLiveStream({
  sampleRate: 16000,
  onChunk: async (samples) => {
    const { result, isEndpoint } = await stream.processAudioChunk(samples, 16000);
    if (result.text) console.log('Partial:', result.text);
    if (isEndpoint) {
      console.log('Final:', result.text);
      await stream.reset();  // Prepare for next utterance
    }
  },
});

await mic.start();
// ... later ...
await mic.stop();
await stream.release();
await engine.destroy();
```

## Typical Recognition Loop

```typescript
const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/streaming-zipformer-en' },
  modelType: 'transducer',
  enableEndpoint: true,
});

const stream = await engine.createStream();

async function onAudioChunk(samples: number[], sampleRate: number) {
  await stream.acceptWaveform(samples, sampleRate);
  
  // Decode all ready chunks
  while (await stream.isReady()) {
    await stream.decode();
    const result = await stream.getResult();
    
    if (result.text) {
      updateUI(result.text);  // Show partial result
    }
    
    if (await stream.isEndpoint()) {
      onUtteranceEnd(result.text);  // Finalize
      await stream.reset();  // Reuse for next utterance
    }
  }
}

// When recording stops
await stream.inputFinished();
// Decode remaining audio
while (await stream.isReady()) {
  await stream.decode();
  const result = await stream.getResult();
  console.log('Final:', result.text);
}

await stream.release();
await engine.destroy();
```

## Input Normalization

By default, `processAudioChunk()` applies **adaptive input normalization**: it scales each chunk so the peak is about 0.8. This helps with varying device input levels (e.g. quiet mics on iOS).

Disable if your audio is already normalized:

```typescript
const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/streaming-zipformer' },
  modelType: 'transducer',
  enableInputNormalization: false,  // pass audio through unchanged
});
```

## Hotwords (Transducer)

For transducer models, you can pass a hotwords file at init and/or a hotwords string per stream:

```typescript
const engine = await createStreamingSTT({
  modelPath: { type: 'asset', path: 'models/streaming-zipformer-en' },
  modelType: 'transducer',
  hotwordsFile: '/path/to/hotwords.txt',
  hotwordsScore: 1.5,
});

const stream = await engine.createStream('optional inline hotwords');
```

## Multiple Streams

You can create multiple streams from one engine (e.g. for different channels):

```typescript
const engine = await createStreamingSTT({ /* ... */ });
const streamA = await engine.createStream();
const streamB = await engine.createStream();

// Use streamA and streamB independently

await streamA.release();
await streamB.release();
await engine.destroy();
```

## Supported Streaming Models

| Type | Files | Notes |
|------|-------|-------|
| `transducer` | encoder.onnx, decoder.onnx, joiner.onnx, tokens.txt | Zipformer, streaming-capable transducers |
| `paraformer` | encoder.onnx, decoder.onnx, tokens.txt | Streaming paraformer |
| `zipformer2_ctc` | model.onnx, tokens.txt | CTC-based |
| `nemo_ctc` | model.onnx, tokens.txt | NeMo CTC |
| `tone_ctc` | model.onnx, tokens.txt | Tone CTC (e.g. Russian T-ONE) |

<Warning>
  **Offline-only models** (e.g. Whisper, SenseVoice) do **not** support streaming. Use `getOnlineTypeOrNull()` to check before creating a streaming engine.
</Warning>

## Complete Example

```typescript
import { createStreamingSTT } from 'react-native-sherpa-onnx/stt';
import { createPcmLiveStream } from 'react-native-sherpa-onnx/audio';

async function startLiveTranscription() {
  // 1. Create streaming engine
  const engine = await createStreamingSTT({
    modelPath: { type: 'asset', path: 'models/sherpa-onnx-streaming-zipformer-en-2023-06-26' },
    modelType: 'transducer',
    enableEndpoint: true,
    numThreads: 2,
  });
  
  // 2. Create stream
  const stream = await engine.createStream();
  
  let transcript = '';
  
  // 3. Start microphone
  const mic = await createPcmLiveStream({
    sampleRate: 16000,
    onChunk: async (samples) => {
      const { result, isEndpoint } = await stream.processAudioChunk(samples, 16000);
      
      if (result.text) {
        transcript = result.text;
        console.log('Partial:', transcript);
      }
      
      if (isEndpoint) {
        console.log('Final:', transcript);
        transcript = '';
        await stream.reset();
      }
    },
  });
  
  await mic.start();
  console.log('Recording started. Speak now...');
  
  // Stop after 10 seconds
  setTimeout(async () => {
    await mic.stop();
    await stream.inputFinished();
    
    // Get final result
    while (await stream.isReady()) {
      await stream.decode();
      const result = await stream.getResult();
      if (result.text) console.log('Final:', result.text);
    }
    
    await stream.release();
    await engine.destroy();
  }, 10000);
}
```
