## Status

Speech enhancement support is **planned for a future release**. This feature will enable:

- Noise reduction and removal
- Audio quality improvement
- Echo and reverberation cancellation
- Background sound suppression
- Pre-processing for better STT accuracy

## Planned API

The enhancement module will provide the following functionality:

```typescript
import { initializeEnhancement, enhanceAudio } from 'react-native-sherpa-onnx/enhancement';

// Initialize enhancement with model
await initializeEnhancement({
  modelPath: { type: 'asset', path: 'models/enhancement-model' },
});

// Enhance noisy audio
const result = await enhanceAudio('/path/to/noisy-audio.wav');
console.log('Enhanced audio saved to:', result.outputPath);

// Clean up
await unloadEnhancement();
```

## Integration with STT

When available, enhancement will work seamlessly as a pre-processing step:

```typescript
// Future combined workflow
import { enhanceAudio } from 'react-native-sherpa-onnx/enhancement';
import { createSTT } from 'react-native-sherpa-onnx/stt';

// 1. Enhance noisy audio
const enhanced = await enhanceAudio('/path/to/noisy-audio.wav');

// 2. Transcribe enhanced audio
const stt = await createSTT({ /* ... */ });
const result = await stt.transcribeFile(enhanced.outputPath);

console.log('Transcript (from enhanced audio):', result.text);

await stt.destroy();
```

## Sherpa ONNX Enhancement Status

The upstream [sherpa-onnx](https://github.com/k2-fsa/sherpa-onnx) library's enhancement support is under evaluation. Implementation in this React Native wrapper will follow upstream availability.

Potential enhancement models include:

- Noise suppression models (RNNoise-style)
- Speech separation models
- Acoustic echo cancellation
- Bandwidth extension

## Workarounds

Until enhancement is implemented, you can:

1. **Use external tools:** Pre-process audio with desktop noise reduction tools before importing to mobile
2. **Record in quiet environments:** Encourage users to record in low-noise settings
3. **Use hardware solutions:** Leverage device noise cancellation features during recording
4. **Cloud-based enhancement:** Use cloud APIs for audio enhancement (requires internet)
5. **Manual post-processing:** Edit recordings in audio editing apps before transcription
