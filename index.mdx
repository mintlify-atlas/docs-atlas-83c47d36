<div className="mt-16 mb-16 max-w-5xl mx-auto px-6">
  <h2 className="text-2xl sm:text-3xl font-semibold dark:text-white text-gray-900 mb-4">
    Quick Start
  </h2>
  <p className="text-base dark:text-gray-400 text-gray-600 mb-8">
    Get started with React Native Sherpa ONNX in just a few steps
  </p>

  <Steps>
    <Step title="Install the package">
      Add the library to your React Native project using npm or yarn.

      <CodeGroup>
      ```bash npm
      npm install react-native-sherpa-onnx
      ```

      ```bash yarn
      yarn add react-native-sherpa-onnx
      ```

      ```bash pnpm
      pnpm add react-native-sherpa-onnx
      ```
      </CodeGroup>
    </Step>

    <Step title="Download a model">
      Choose and download a pre-trained model for your use case. Models are available for various languages and tasks.

      ```typescript
      import { ModelDownloadManager } from 'react-native-sherpa-onnx/download';

      const manager = new ModelDownloadManager();
      await manager.downloadModel('whisper-tiny-en');
      ```
    </Step>

    <Step title="Start processing speech">
      Create an STT or TTS engine and start processing audio.

      ```typescript
      import { createSTT } from 'react-native-sherpa-onnx/stt';

      const stt = await createSTT({
        modelPath: { type: 'asset', path: 'models/whisper-tiny-en' }
      });

      const result = await stt.transcribeFile('/path/to/audio.wav');
      console.log(result.text);
      ```
    </Step>
  </Steps>
</div>
