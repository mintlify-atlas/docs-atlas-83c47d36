# Quick Start

This guide will walk you through building a simple React Native app that performs speech-to-text transcription and text-to-speech generation. You'll learn the core concepts and APIs in just a few minutes.

<Note>
Make sure you've completed the [installation](/installation) steps before proceeding.
</Note>

## Overview

In this quickstart, you'll:

1. Download a pre-trained model
2. Set up a basic React Native component
3. Perform speech-to-text transcription
4. Generate speech from text
5. Stream real-time audio transcription

## Step 1: Download a Model

<Steps>
  <Step title="Choose a model type">
    For this quickstart, we'll use Whisper Tiny for STT (speech-to-text) and a VITS model for TTS (text-to-speech).

    **STT Model Options:**
    - **Whisper Tiny**: Fast, multilingual, good for testing (~39 MB)
    - **Paraformer**: Very fast for production use
    - See [all STT models](https://k2-fsa.github.io/sherpa/onnx/pretrained_models/index.html)

    **TTS Model Options:**
    - **VITS-Piper**: High quality, fast synthesis
    - **Kokoro**: Multi-speaker, multi-language
    - See [all TTS models](https://github.com/k2-fsa/sherpa-onnx/releases/tag/tts-models)
  </Step>

  <Step title="Download the STT model">
    Download Whisper Tiny from [sherpa-onnx releases](https://k2-fsa.github.io/sherpa/onnx/pretrained_models/whisper/tiny.en.html):

    ```bash
    # Download and extract
    wget https://github.com/k2-fsa/sherpa-onnx/releases/download/asr-models/sherpa-onnx-whisper-tiny.en.tar.bz2
    tar xvf sherpa-onnx-whisper-tiny.en.tar.bz2
    ```
  </Step>

  <Step title="Download the TTS model">
    Download a VITS-Piper model:

    ```bash
    wget https://github.com/k2-fsa/sherpa-onnx/releases/download/tts-models/vits-piper-en_US-amy-low.tar.bz2
    tar xvf vits-piper-en_US-amy-low.tar.bz2
    ```
  </Step>

  <Step title="Bundle models with your app">
    Place the extracted model directories in your React Native project:

    **Android:**
    ```
    android/app/src/main/assets/models/
    ├── sherpa-onnx-whisper-tiny.en/
    └── vits-piper-en_US-amy-low/
    ```

    **iOS:**
    1. Drag the model folders into Xcode project navigator
    2. Check "Create folder references" (not "Create groups")
    3. Ensure they're added to the app target

    <Tip>
    Alternatively, download models at runtime using the [Model Download Manager](/core/model-download).
    </Tip>
  </Step>
</Steps>

## Step 2: Create Your Component

Let's create a simple component that demonstrates both STT and TTS:

```tsx App.tsx
import React, { useState, useEffect } from 'react';
import { View, Text, Button, StyleSheet } from 'react-native';
import { createSTT, createTTS } from 'react-native-sherpa-onnx/stt';
import { createTTS as initTTS } from 'react-native-sherpa-onnx/tts';
import type { SttEngine, TtsEngine } from 'react-native-sherpa-onnx/stt';

function App() {
  const [sttEngine, setSttEngine] = useState<SttEngine | null>(null);
  const [ttsEngine, setTtsEngine] = useState<TtsEngine | null>(null);
  const [status, setStatus] = useState('Initializing...');
  const [result, setResult] = useState('');

  // Initialize engines on mount
  useEffect(() => {
    initializeEngines();
    return () => {
      // Clean up on unmount
      sttEngine?.destroy();
      ttsEngine?.destroy();
    };
  }, []);

  const initializeEngines = async () => {
    try {
      // Initialize STT
      const stt = await createSTT({
        modelPath: {
          type: 'asset',
          path: 'models/sherpa-onnx-whisper-tiny.en',
        },
        modelType: 'whisper',
      });
      setSttEngine(stt);

      // Initialize TTS
      const tts = await initTTS({
        modelPath: {
          type: 'asset',
          path: 'models/vits-piper-en_US-amy-low',
        },
        modelType: 'vits',
      });
      setTtsEngine(tts);

      setStatus('Ready!');
    } catch (error) {
      setStatus(`Error: ${error.message}`);
    }
  };

  return (
    <View style={styles.container}>
      <Text style={styles.title}>React Native Sherpa ONNX</Text>
      <Text style={styles.status}>{status}</Text>
      {result ? <Text style={styles.result}>{result}</Text> : null}
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    padding: 20,
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    marginBottom: 20,
  },
  status: {
    fontSize: 16,
    color: '#666',
    marginBottom: 20,
  },
  result: {
    fontSize: 18,
    color: '#333',
    textAlign: 'center',
  },
});

export default App;
```

## Step 3: Speech-to-Text Transcription

Now let's add STT functionality to transcribe an audio file:

```tsx Transcription Example
import { createSTT } from 'react-native-sherpa-onnx/stt';
import type { SttEngine, SttRecognitionResult } from 'react-native-sherpa-onnx/stt';

// In your component:
const transcribeAudio = async () => {
  if (!sttEngine) {
    console.log('STT engine not initialized');
    return;
  }

  setStatus('Transcribing...');

  try {
    // Transcribe an audio file
    const result: SttRecognitionResult = await sttEngine.transcribeFile(
      '/path/to/your/audio.wav'
    );

    // Display the transcription
    setResult(result.text);
    setStatus('Transcription complete!');

    // Access additional information
    console.log('Tokens:', result.tokens);
    console.log('Timestamps:', result.timestamps);
    console.log('Language:', result.lang);
    console.log('Emotion:', result.emotion); // For SenseVoice models
  } catch (error) {
    setStatus(`Transcription failed: ${error.message}`);
  }
};

// Add button to your JSX:
<Button title="Transcribe Audio" onPress={transcribeAudio} />
```

### Transcribe PCM Samples

You can also transcribe raw audio samples directly:

```typescript
// Transcribe from PCM samples (float32 array, mono)
const samples = new Float32Array(16000); // 1 second of audio at 16kHz
const sampleRate = 16000;

const result = await sttEngine.transcribeSamples(
  Array.from(samples),
  sampleRate
);

console.log('Transcription:', result.text);
```

## Step 4: Text-to-Speech Generation

Add TTS functionality to generate speech from text:

```tsx TTS Example
import { createTTS, saveAudioToFile } from 'react-native-sherpa-onnx/tts';
import type { TtsEngine, GeneratedAudio } from 'react-native-sherpa-onnx/tts';
import { DocumentDirectoryPath } from '@dr.pogodin/react-native-fs';

// In your component:
const generateSpeech = async () => {
  if (!ttsEngine) {
    console.log('TTS engine not initialized');
    return;
  }

  setStatus('Generating speech...');

  try {
    // Generate speech from text
    const audio: GeneratedAudio = await ttsEngine.generateSpeech(
      'Hello! Welcome to React Native Sherpa ONNX.',
      {
        speed: 1.0,  // Normal speed
        sid: 0,      // Speaker ID (for multi-speaker models)
      }
    );

    console.log(`Generated ${audio.samples.length} samples`);
    console.log(`Sample rate: ${audio.sampleRate}Hz`);

    // Save to WAV file
    const outputPath = `${DocumentDirectoryPath}/output.wav`;
    await saveAudioToFile(audio, outputPath);

    setStatus(`Speech saved to ${outputPath}`);
    setResult(`Generated ${audio.samples.length} audio samples`);

    // Now you can play the audio file using react-native-sound or similar
  } catch (error) {
    setStatus(`Generation failed: ${error.message}`);
  }
};

// Add button to your JSX:
<Button title="Generate Speech" onPress={generateSpeech} />
```

### Generate with Timestamps

Get word-level timing information:

```typescript
const audioWithTimestamps = await ttsEngine.generateSpeechWithTimestamps(
  'This is a test sentence.'
);

console.log('Subtitles:', audioWithTimestamps.subtitles);
// Output: [{ text: 'This', start: 0.0, end: 0.2 }, ...]
```

## Step 5: Real-Time Streaming STT

For live microphone transcription, use the streaming API:

```tsx Streaming STT Example
import { createStreamingSTT } from 'react-native-sherpa-onnx/stt';
import { createPcmLiveStream } from 'react-native-sherpa-onnx/audio';
import type {
  StreamingSttEngine,
  SttStream,
} from 'react-native-sherpa-onnx/stt';

// In your component:
const [streamingEngine, setStreamingEngine] = useState<StreamingSttEngine | null>(null);
const [isRecording, setIsRecording] = useState(false);
const [partialResult, setPartialResult] = useState('');

const initStreaming = async () => {
  try {
    const engine = await createStreamingSTT({
      modelPath: {
        type: 'asset',
        path: 'models/sherpa-onnx-whisper-tiny.en',
      },
      modelType: 'transducer', // Use streaming-compatible model
      enableEndpoint: true,
    });
    setStreamingEngine(engine);
  } catch (error) {
    console.error('Failed to initialize streaming:', error);
  }
};

const startRecording = async () => {
  if (!streamingEngine) {
    console.log('Streaming engine not initialized');
    return;
  }

  setIsRecording(true);

  // Create a stream for this recording session
  const stream = await streamingEngine.createStream();

  // Start capturing microphone audio
  const liveStream = createPcmLiveStream(
    16000, // Sample rate
    1,     // Channels (mono)
    512    // Samples per callback
  );

  // Process audio data
  liveStream.onData((data) => {
    // Feed samples to the recognizer
    stream.acceptWaveform(data.samples, data.sampleRate);

    // Get partial results in real-time
    const result = streamingEngine.getResult(stream);
    setPartialResult(result.text);

    // Check if endpoint detected (end of speech)
    if (streamingEngine.isEndpoint(stream)) {
      const finalResult = streamingEngine.getResult(stream);
      console.log('Final transcription:', finalResult.text);
      streamingEngine.reset(stream);
    }
  });

  liveStream.onError((error) => {
    console.error('Recording error:', error);
  });

  // Start recording
  await liveStream.start();
};

const stopRecording = async () => {
  setIsRecording(false);
  // Stop the live stream and get final result
};

// Add buttons to your JSX:
<Button
  title={isRecording ? 'Stop Recording' : 'Start Recording'}
  onPress={isRecording ? stopRecording : startRecording}
/>
{partialResult ? (
  <Text style={styles.partial}>Partial: {partialResult}</Text>
) : null}
```

<Warning>
Make sure to use a **streaming-compatible model** (transducer, paraformer, nemo_ctc, zipformer2_ctc, or tone_ctc) for real-time recognition. Whisper is not suitable for streaming.
</Warning>

## Complete Example

Here's a complete working example combining STT and TTS:

```tsx Complete App Example
import React, { useState, useEffect } from 'react';
import {
  View,
  Text,
  Button,
  StyleSheet,
  ScrollView,
  ActivityIndicator,
} from 'react-native';
import { createSTT } from 'react-native-sherpa-onnx/stt';
import { createTTS, saveAudioToFile } from 'react-native-sherpa-onnx/tts';
import type { SttEngine } from 'react-native-sherpa-onnx/stt';
import type { TtsEngine } from 'react-native-sherpa-onnx/tts';
import { DocumentDirectoryPath } from '@dr.pogodin/react-native-fs';

function App() {
  const [sttEngine, setSttEngine] = useState<SttEngine | null>(null);
  const [ttsEngine, setTtsEngine] = useState<TtsEngine | null>(null);
  const [loading, setLoading] = useState(true);
  const [status, setStatus] = useState('Initializing engines...');
  const [transcription, setTranscription] = useState('');

  useEffect(() => {
    initialize();
    return cleanup;
  }, []);

  const initialize = async () => {
    try {
      // Initialize STT
      const stt = await createSTT({
        modelPath: { type: 'asset', path: 'models/sherpa-onnx-whisper-tiny.en' },
        modelType: 'whisper',
        numThreads: 2,
      });
      setSttEngine(stt);

      // Initialize TTS
      const tts = await createTTS({
        modelPath: { type: 'asset', path: 'models/vits-piper-en_US-amy-low' },
        modelType: 'vits',
        numThreads: 2,
      });
      setTtsEngine(tts);

      setStatus('Ready!');
      setLoading(false);
    } catch (error) {
      setStatus(`Initialization failed: ${error.message}`);
      setLoading(false);
    }
  };

  const cleanup = () => {
    sttEngine?.destroy();
    ttsEngine?.destroy();
  };

  const handleTranscribe = async () => {
    if (!sttEngine) return;

    setStatus('Transcribing...');
    try {
      // Replace with your audio file path
      const result = await sttEngine.transcribeFile(
        `${DocumentDirectoryPath}/test_audio.wav`
      );
      setTranscription(result.text);
      setStatus('Transcription complete!');
    } catch (error) {
      setStatus(`Error: ${error.message}`);
    }
  };

  const handleGenerateSpeech = async () => {
    if (!ttsEngine || !transcription) return;

    setStatus('Generating speech...');
    try {
      const audio = await ttsEngine.generateSpeech(transcription, {
        speed: 1.0,
      });

      const outputPath = `${DocumentDirectoryPath}/generated_speech.wav`;
      await saveAudioToFile(audio, outputPath);

      setStatus(`Speech saved to ${outputPath}`);
    } catch (error) {
      setStatus(`Error: ${error.message}`);
    }
  };

  if (loading) {
    return (
      <View style={styles.container}>
        <ActivityIndicator size="large" />
        <Text style={styles.status}>{status}</Text>
      </View>
    );
  }

  return (
    <ScrollView style={styles.container}>
      <View style={styles.content}>
        <Text style={styles.title}>Sherpa ONNX Demo</Text>
        <Text style={styles.status}>{status}</Text>

        <Button title="Transcribe Audio File" onPress={handleTranscribe} />

        {transcription ? (
          <View style={styles.resultContainer}>
            <Text style={styles.label}>Transcription:</Text>
            <Text style={styles.result}>{transcription}</Text>
            <Button
              title="Generate Speech from Text"
              onPress={handleGenerateSpeech}
            />
          </View>
        ) : null}
      </View>
    </ScrollView>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#fff',
  },
  content: {
    padding: 20,
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    marginBottom: 20,
    textAlign: 'center',
  },
  status: {
    fontSize: 16,
    color: '#666',
    marginBottom: 20,
    textAlign: 'center',
  },
  resultContainer: {
    marginTop: 20,
    padding: 15,
    backgroundColor: '#f5f5f5',
    borderRadius: 8,
  },
  label: {
    fontSize: 14,
    fontWeight: 'bold',
    color: '#333',
    marginBottom: 8,
  },
  result: {
    fontSize: 16,
    color: '#000',
    marginBottom: 15,
  },
});

export default App;
```

## Common Issues

<AccordionGroup>
  <Accordion title="Model files not found">
    Make sure the model files are correctly bundled:

    - **Android**: Place in `android/app/src/main/assets/models/`
    - **iOS**: Add to Xcode as folder references (not groups)

    Verify with:
    ```typescript
    import { listAssetModels } from 'react-native-sherpa-onnx';
    const models = await listAssetModels();
    console.log('Available models:', models);
    ```
  </Accordion>

  <Accordion title="Out of memory errors">
    Large models (like Whisper Large) may cause OOM on devices with limited RAM.

    Solutions:
    - Use smaller models (Tiny, Base instead of Large)
    - Use quantized (int8) models
    - Reduce `numThreads` in initialization options
  </Accordion>

  <Accordion title="Slow inference">
    To improve performance:

    - Use hardware acceleration (NNAPI, Core ML, XNNPACK)
    - Choose faster model types (Paraformer, transducer)
    - Increase `numThreads` (default is 2)
    - Use quantized models

    ```typescript
    const stt = await createSTT({
      modelPath: { type: 'asset', path: 'models/...' },
      numThreads: 4,
      provider: 'nnapi', // Android only
    });
    ```
  </Accordion>
</AccordionGroup>
